Browser based camera test

Simple web interface for testing camera access on desktop and mobile browsers.

Home Engineering projects

Overview

This project is a minimal HTML and JavaScript application that opens the camera in modern desktop and mobile browsers. It demonstrates the use of navigator.mediaDevices.getUserMedia for video capture and supports basic start and stop controls.

Key features

  • Single button to start the camera stream.
  • Works on Windows and Linux desktop browsers and mobile browsers.
  • Uses only standard web APIs and can run from a static host.
